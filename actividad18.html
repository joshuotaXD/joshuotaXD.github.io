Descripción de la Actividad

Análisis de la matriz de adyacencia como una forma tabular de representación de grafos que permite identificar conexiones directas mediante el uso de valores booleanos o numéricos en una estructura bidimensional

Matriz de Adyacencia

La Matriz de Adyacencia es una estructura de datos bidimensional donde tanto las filas como las columnas representan los nodos del grafo permitiendo verificar de forma inmediata si existe un vínculo entre dos vértices específicos mediante la intersección de sus índices

   A B C
A 0 1 1
B 1 0 0
C 1 0 0

Dentro de este esquema el valor 1 indica la existencia de una arista mientras que el 0 representa la ausencia de conexión siendo una herramienta muy sencilla de implementar pero con la desventaja de requerir un espacio de memoria cuadrático lo que la vuelve poco eficiente para grafos con pocos enlaces
