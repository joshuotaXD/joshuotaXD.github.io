Pregunta clase 5

¿Cómo creen que podrían organizar mejor la información en su repositorio web para que sea más clara y fácil de actualizar?

Para mejorar la organización, es fundamental implementar una estructura de carpetas semántica (separando CSS, imágenes y archivos por parciales), utilizar comentarios descriptivos en el código y mantener un archivo README actualizado que sirva como guía del proyecto.

Ventajas de un Editor de Código y sus Funciones

  • Resaltado de Sintaxis: Facilita la lectura al diferenciar etiquetas, atributos y valores mediante colores.
  • Autocompletado (IntelliSense): Ayuda a escribir código más rápido y reduce errores de escritura en etiquetas o rutas.
  • Formateo Automático: Permite mantener el código indentado y ordenado con un solo clic, mejorando la jerarquía visual.
  • Extensiones y Git Integrado: Permite gestionar cambios en el repositorio y añadir herramientas de productividad directamente desde el editor.
  • Edición Multilínea: Facilita realizar cambios repetitivos en varias líneas de código al mismo tiempo, ahorrando tiempo valioso.
