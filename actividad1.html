En esta actividad exploramos los conceptos fundamentales que sirven como base para la programación y el manejo de sistemas.

Diferencia entre Dato e Información

Un Dato es simplemente un valor base o un símbolo básico como un número o una letra que carece de contexto por lo cual no transmite un mensaje por sí mismo mientras que la Información es el resultado de procesar y organizar esos datos para darles un significado útil que permite tomar decisiones o entender una situación específica

Concepto de Estructura de Datos.

Se define como un método o formato especializado para organizar y almacenar datos dentro de un sistema informático de manera que se puedan realizar operaciones con ellos de forma eficiente además de facilitar el acceso y la modificación de los valores según las necesidades del programa o el algoritmo que se esté ejecutando
