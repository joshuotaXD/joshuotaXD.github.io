TIPOS DE DATOS Y ALGORITMOS

En programación los tipos de datos definen el tipo de valor que una variable puede almacenar y qué operaciones se pueden realizar con ellos encontrando comúnmente los enteros para números sin decimales los flotantes para valores con punto decimal los booleanos que solo representan verdadero o falso y las cadenas de texto que agrupan caracteres alfanuméricos

ALGORITMOS

Un algoritmo se define como una secuencia lógica y finita de pasos o instrucciones bien definidas que se siguen para resolver un problema específico o realizar una tarea determinada funcionando como una receta que guía a la computadora desde un estado inicial hasta un resultado final de manera precisa
